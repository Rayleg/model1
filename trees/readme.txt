Этот классификатор решает задачу классификации по возрасту
Инструкция:
	Получить набор признаков по умолчанию
cols = trees.choose_targets()
	Получить векторы из входных данных(по умолчанию data/data.csv) вместе с названиями
x, y, names = trees.read_data_set(cols)
	Можно вывести статистику по векторам
trees.print_set_stats(x, y, names)
	Построить классификатор с параметрами по умолчанию
cls = trees.build_classifier()
	Обучение классификатора
cls = cls.fit(x, y)
	Дерево можно отрисовать в файл по умолчанию
cls.tree.draw_me(names)
	Можно использовать метрику R^2 для оценки, даже на исходных данных:
cls.score(x, y)
	Доступны pruning, predict
